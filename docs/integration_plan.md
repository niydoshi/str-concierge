# Integration Plan

## Current API Integration

The current prototype uses an AI API to generate guest-friendly responses from a property knowledge base.

The app:
1. Loads property-specific information from a local markdown file
2. Accepts a guest question
3. Sends the question and property context to the AI API
4. Returns a suggested guest response

## Future Integrations

Future versions could connect to:
- Hostfully property and reservation data
- HostBuddy AI workflows
- Direct booking website content
- Guest messaging channels
